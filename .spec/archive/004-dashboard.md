# Spec: Главная панель управления (Dashboard)

## 1. Цель
Создать удобную форму для запуска процессов импорта и анализа без входа в редактор VBA.

## 2. Элементы формы (UI Controls)
Форма: `uf_Dashboard` (Свободная, не привязанная к таблице).

| Имя контрола | Тип | Подпись (Caption) | Действие |
|---|---|---|---|
| `btnImport` | Кнопка | 1. Загрузить Excel | Запуск `mod_Import_Logic.ImportExcelData` |
| `btnAnalyze` | Кнопка | 2. Анализ изменений | Запуск `mod_Analysis_Logic.SyncBufferToMaster` |
| `btnOpenLog` | Кнопка | 3. Журнал событий | Открытие таблицы `tbl_History_Log` |
| `lblStatus` | Надпись | (Пусто) | Вывод сообщений о статусе |

## 3. Логика (Code Behind)
- При нажатии кнопок вызывать соответствующие функции.
- Обрабатывать ошибки.
- Выводить "Готово" в `lblStatus`.
