# Spec: Логика импорта данных (Excel -> Access)

## 1. Цель
Реализовать безопасную загрузку данных из Excel-файла (формат "Алушта"/1С) в таблицу `tbl_Import_Buffer`.

## 2. Проблема (Mapping)
Заголовки в Excel (русские, с пробелами) не совпадают с именами полей в Access (русские, с подчеркиваниями).
- Excel: "Личный номер" (с пробелом)
- Access: `Личный_номер` (с подчеркиванием)

## 3. Техническое решение
Мы не будем использовать прямой импорт, так как порядок столбцов может меняться.
**Алгоритм:**
1. Показать диалоговое окно выбора файла (`msoFileDialogFilePicker`).
2. Очистить таблицу `tbl_Import_Buffer`.
3. Подключить Excel-файл как **Связанную таблицу** (Linked Table) с именем `tmp_Excel_Link`.
4. Выполнить SQL-запрос `INSERT INTO ... SELECT ...`, который явно переложит данные из русских колонок Excel (с пробелами в квадратных скобках) в поля Access (с подчеркиваниями).
5. Удалить связь с таблицей `tmp_Excel_Link`.

## 4. Карта полей (Mapping)
| Access Field (Buffer) | Excel Column Header | Примечание |
|---|---|---|
| Лицо | [Лицо] | Первый столбец (ID, числовой) |
| Личный_номер | [Личный номер] | |
| Воинское_звание | [Воинское звание] | |
| Лицо_ФИО | [Лицо] или [ФИО] или [Лицо_1] | **Четвертый столбец** (ФИО, текст) |
| Дата_рождения | [Дата рождения] | |
| Статус_занятости | [Статус занятости] | |
| Штатная_должность | [Штатная должность] | |
| Должность | [Должность] | |
| Дата_приказа_ЛС | [Дата приказа Л/С] | |
| Номер_приказа_ЛС | [Номер приказа] | |

### 4.1. Обработка дубликатов заголовков
В Excel есть два столбца с названием "Лицо":
- **Столбец 0 (первый)**: Числовой ID (например, 163676)
- **Столбец 3 (четвертый)**: ФИО (например, "Иванов Алексей Александрович")

**Рекомендация:** Переименовать столбец ФИО в Excel в "ФИО" перед импортом для упрощения и надежности.

**Альтернатива:** Код автоматически определяет столбец ФИО по:
1. Поиску столбца с названием "ФИО" (если переименован)
2. Позиции (4-й столбец, индекс 3)
3. Автоматическим переименованиям Access (например, "Лицо_1", "Лицо_2")

## 5. План реализации (Files)
1. **`mod_Import_Logic.bas`**:
   - `Function SelectExcelFile() As String`: Выбор файла через диалог.
   - `Function CreateExcelLink(strPath As String) As Boolean`: Создание связанной таблицы.
   - `Sub DeleteExcelLink()`: Удаление связи.
   - `Sub ClearImportBuffer()`: Очистка буфера.
   - `Function ImportExcelData(strPath As String) As Boolean`: Основная логика импорта.

2. **SQL (внутри VBA)**:
   ```sql
   INSERT INTO tbl_Import_Buffer (Личный_номер, Лицо_ФИО, ...)
   SELECT [Личный номер], [Лицо], ...
   FROM tmp_Excel_Link
   WHERE [Личный номер] IS NOT NULL;
   ```

## 6. Критерии приемки
- Диалог выбора файла работает корректно.
- Связанная таблица создается и удаляется без ошибок.
- Данные импортируются с правильным mapping полей.
- Пустые строки (без личного номера) не импортируются.
- Обработка ошибок работает корректно.
